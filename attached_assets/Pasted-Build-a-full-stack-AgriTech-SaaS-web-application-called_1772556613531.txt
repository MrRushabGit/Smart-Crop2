Build a full-stack AgriTech SaaS web application called AgriNova.



This is a production-level Smart Crop Advisory Platform with modern UI, analytics, authentication, and backend integration.



The system must feel like a funded AgriTech startup product.







🌿 CORE CONCEPT





AgriNova is an AI-powered crop advisory web platform where:



Farmers log in

Enter agricultural parameters

See analytics

And all activity is stored for admin analysis

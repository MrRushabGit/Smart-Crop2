🌱 AgriNova — Smart Crop Advisory Platform
AgriNova is a full-stack agri-tech application that helps farmers make data-driven crop decisions based on their field conditions and resource usage. 

🔗 Live Demo
👉 https://smart-crop2-git-main-rushabengollabtech2028-5732s-projects.vercel.app

🚀 Overview
Users input key farm parameters such as land area, water usage, fertilizer, pesticide usage, soil type, irrigation method, and season. The system analyzes this data using real agricultural datasets and provides optimized crop recommendations and insights.

🧠 Features
🔐 Secure user authentication
📊 Personalized crop recommendations
🤖 Data-driven analysis (Kaggle datasets)
🧾 History tracking of user inputs
🎨 Clean and intuitive UI

🏗️ Tech Stack
Frontend: React, Tailwind CSS
Backend: Node.js, Express
Database: PostgreSQL
Deployment: Vercel (Frontend), Render (Backend)

🎯 Impact
AgriNova enables smarter farming by improving crop selection and optimizing resource usage through technology-driven insights.

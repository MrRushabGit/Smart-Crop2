# Smart Crop Advisory 🌱

A machine learning–based project that helps farmers choose the right crop
based on soil, climate, and environmental conditions.

## Features
- Crop prediction using ML
- Data preprocessing with pandas
- Decision Tree with hyperparameter tuning
- Simple and scalable backend

## Tech Stack
- Python
- Pandas
- Scikit-learn
- Git & GitHub
